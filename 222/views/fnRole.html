：原型深入：函数的三种角色

函数的三种角色：

  1. 普通函数

    • 堆栈内存释放（根据有没有被占用）

    • 作用域链（看是不是私有作用域）

  2. 类

    • prototype：原型

    • __proto：原型链

    • 实例

  3. 普通对象

    • 和普通的obj没有啥区别，就是对键值对的增删改查
      三种角色之间没有什么必然关系

function Fn(){

    var n=10;

    this.m=100;

}

Fn.prototype.aa=function(){

    console.log(this.m);

}

Fn.bb=function(){

    console.log('bb');

}

//1. 普通函数

Fn();//第一步：形成作用域；第二步：形参赋值；第三步：变量提升；最后，代码自上而下执行；this:window;有一个私有变量n,和原型以及属性bb没有关系



//2. 构造函数执行

var f=new Fn();//先像普通函数一样执行，只不过this是当前实例

console.log(f.n)//undefined;n是私有变量和实例没有关系

console.log(f.m);//100;实例上的私有属性

f.aa();//实例通过__proto__找到prototype上面的方法

console.log(f.bb);//undefined；bb是把fn当作一个普通对象设置的属性而已，和实例等没有半毛钱关系



//3. 普通对象

Fn.bb();
