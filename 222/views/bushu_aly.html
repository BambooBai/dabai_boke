阿里云部署项目

前言：
linux操作系统分为：centors、debian、ubuntu
一般使用centors较多

一、Git 免密push 配置

第一次连接，登录阿里云控制台-》实例-》登录-》进入后台（免密码git push）

  1. 打开终端

  2. 命令：ls -al ~/.ssh查询有公钥（id_rsa.pub）和私钥（id_rsa）

  3. 如果没有生成，则命令：ssh-keygen -t rsa -C "git邮箱"生成密钥

  4. 命令：cat ~/.ssh/id_rsa.pub查看自己的公钥

  5. 登录github，把公钥放到github上（这时，可以免密码git push）

二、免密码登录阿里云配置

  1. 本地命令：cp ~/.ssh/id_rsa.pub ./ copy创建密钥文件

  2. 把公钥放到push到github上

  3. 在阿里云上git pull密钥文件

  4. 命令：cat id_rsa.pub >> ~/.ssh/authorized_keys 在授权key中追加我的密钥

  5. 完成


三、在本地重命名快捷登录阿里云

  1. 在根目录下vi go.sh(创建go.sh；添加图片命令；按x保存退出)


  2. ls -l详细查看本地文件

    • x是执行权利，当前文件没有执行权利

    • chmod +x go.sh 赋予文件执行权利



  3. ./go.sh 完美登录😯😊

四、部署

  • 把本地项目push到github上

  • 命令：ssh root@远程服务ip 登录远程成功

  • 使用ls查看服务器文件

  • 命令：cd 个人目录

  • 命令git clone 项目地址

  • 安装node 地址未尝试
